### UNIT1: 创建版本库
进入某个文件夹: 注意使用的是斜杠“/”，这一点和windows的路径不一样

`cd /e/5___Study/Python/14___MMRec/jobs/ConformalPrediction`

- 初始化仓库: `git init`

- 添加文件：添加文件指令可以多次执行

    `git add file1.txt`
    `git add file2.txt file3.txt`
- 提交更新： 
  `git commit -m "自定义的更新描述"`
- Tips:
    commit 在git中被称作“快照”，就相当于游戏的存档。
